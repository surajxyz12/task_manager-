# Task Manager

A minimal, keyboard-friendly task manager built with React and Tailwind CSS — styled like a discipline log / command deck, with built-in focus tools.

## Features

- **Task management** — add, edit, delete, and mark tasks complete
- **Priority levels** — tag tasks as Low, Medium, or High with a color-coded priority rail
- **Filtering** — view tasks by priority (All / High / Medium / Low)
- **Multi-line tasks** — Shift+Enter for a new line, Enter to submit
- **Timer & Pomodoro** — built-in stopwatch and 25-minute focus timer
- **Analytics** — completed vs. pending breakdown by priority, viewable by day/week/month/year
- **Calendar** — see which dates have tasks logged, with a per-day task view
- **Quick notes** — a lightweight notepad in the header, auto-saved
- **Profile** — editable name, username, and photo
- **Persistent storage** — everything (tasks, notes, profile) saves to `localStorage`, so your data survives a refresh

## Tech Stack

- React (Vite)
- Tailwind CSS v4
- Google Fonts — Barlow Condensed, Inter, JetBrains Mono

## Getting Started

```bash
npm install
npm run dev
```

Then open `http://localhost:5173` in your browser.
